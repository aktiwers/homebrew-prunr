# homebrew-prunr

Homebrew tap for [Prunr](https://github.com/aktiwers/prunr) — local AI background removal.

## Install

```sh
brew install aktiwers/prunr/prunr
```

Or tap first, then install:

```sh
brew tap aktiwers/prunr
brew install prunr
```

## Update

```sh
brew update
brew upgrade prunr
```

## Uninstall

```sh
brew uninstall prunr
brew untap aktiwers/prunr
```

## Supported platforms

- macOS on Apple Silicon (arm64)

For Intel Macs, Linux, or Windows, see the [main repo](https://github.com/aktiwers/prunr) for direct downloads.
